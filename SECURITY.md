# Security Policy

To get more information about security regarding this project. Join our [discord](https://discord.com/invite/fQkQn5ajFR) 

## Supported Versions

We will respond to any vulnerability in the project and make a case by case evaluation about the need to patch major or minor versions. In most cases we will only patch the current major version and the current and prior minor versions of the project.

## Reporting a Vulnerability

To report a vulneratbility please use the confidential reporting tool provided by [GitHub](https://github.com/workfloworchestrator/orchestrator-core/security). We will get back to you A.S.A.P to let you know whether we accept or reject the issue.
